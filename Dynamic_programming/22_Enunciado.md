# 22. Generate Parentheses

Dado um número n de pares de parênteses, escreva uma função para gerar todas as combinações de parênteses bem formados.

Exemplo 1:

Entrada: n = 3
Saída: ["((()))","(()())","(())()","()(())","()()()"]

Exemplo 2:

Entrada: n = 1
Saída: ["()"]

Restrições:

1 <= n <= 8
